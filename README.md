# 🚗 Smart Parking Management System (Java & DSA)

This project is a comprehensive Java application developed to solve real-world parking issues using optimized **Data Structures and Algorithms**.

## 🧠 Data Structures Used
To ensure high performance, I have implemented:
* **Circular Linked List:** For managing parking slots in a continuous loop.
* **Waiting Queue:** To handle vehicles in FIFO order when the lot is full.
* **Heap (Priority Queue):** For VIP and Emergency vehicle priority allocation.
* **Graph:** To visualize the parking layout and find shortest navigation paths.

## 🛠 Features
* Real-time entry and exit logging.
* Automated fee calculation.
* Priority-based slot assignment.
* Clean and modular Java code.

* **Language:** Java
* **Logic:** DSA (Singly/Circular Linked Lists, Heaps, Graphs)
